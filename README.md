# To-Do Web App

A simple and interactive To-Do Web Application developed using HTML, CSS, and JavaScript. This project helps users organize and manage their daily tasks efficiently.

## Objective

The objective of this project is to create a task management application where users can add, complete, and delete tasks while learning the fundamentals of web development.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* User-friendly interface
* Dynamic task management using JavaScript

## Technologies Used

* HTML
* CSS
* JavaScript

## Files

* todo.html

## How to Run

1. Download or clone the project.
2. Open `todo.html` in any web browser.
3. Add, complete, and manage your tasks.

## Outcome

This project helped improve my understanding of HTML, CSS, and JavaScript by implementing real-time user interactions. It also enhanced my skills in DOM manipulation, event handling, and building interactive web applications.
